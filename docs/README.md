# AlphaBot Dashboard

Private automated crypto trading bot dashboard.

## Live version

Runs locally via `serve.py` on port 8765.  
Access remotely via SSH tunnel.

## This GitHub Pages version

Static UI reference only — no live data.  
All WebSocket and API calls require the local server running.

## Stack

- React (inline Babel, no build step)
- LightweightCharts + ECharts
- Python aiohttp backend
- Coinbase Advanced Trade API
